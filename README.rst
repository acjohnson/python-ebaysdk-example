ebaysdk Example
===============

Use this example to list eBay items with Python!

Installation
------------

.. code-block:: shell

    # sudo apt-get install python-virtualenv
    # virtualenv -p /usr/bin/python2.7 venv
    # source venv/bin/activate
    # pip install ebaysdk

Configuration
-------------

Fill out the `ebay.yaml` file.
